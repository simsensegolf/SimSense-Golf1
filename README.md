# SimSense Golf

Web app that turns golf simulator numbers into simple explanations, swing insights, and improvement recommendations.

## Current version

This version includes:
- driver-only analyzer
- current driver setup inputs
- head loft, current loft setting, and hosel setting
- deeper shaft recommendation direction
- analyze button triggers the result set
- green-branded design update
- logo in header and browser tab

## Run locally

```bash
npm install
npm run dev
```

## Deploy updates

Push changes to GitHub and Vercel should redeploy automatically.
